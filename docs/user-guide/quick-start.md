# Inicio Rapido

Guia para comenzar a usar Uniformes System v2.0.

---

## Acceso al Sistema

### 1. Iniciar Sesion

1. Abrir la aplicacion Uniformes System
2. Ingresar usuario y contrasena
3. Click en "Iniciar Sesion"

### 2. Seleccionar Colegio

Despues de iniciar sesion:
1. Usar el selector de colegio en la barra superior
2. Seleccionar el colegio con el que desea trabajar
3. El sistema cargara los productos de ese colegio

---

## Navegacion Principal

| Seccion | Descripcion |
|---------|-------------|
| **Dashboard** | Estadisticas y resumen |
| **Ventas** | Punto de venta (POS) |
| **Productos** | Catalogo y inventario |
| **Clientes** | Gestion de clientes |
| **Pedidos** | Ordenes pendientes |
| **Contabilidad** | Finanzas del negocio |
| **Reportes** | Estadisticas y exportacion |

---

## Realizar una Venta Rapida

1. Ir a **Ventas**
2. Seleccionar productos del catalogo
3. Ajustar tallas y cantidades
4. Click en "Agregar al carrito"
5. Seleccionar cliente (o crear nuevo)
6. Elegir metodo de pago
7. Click en "Completar Venta"
8. Imprimir recibo (opcional)

---

## Metodos de Pago

| Metodo | Descripcion |
|--------|-------------|
| Efectivo | Pago en caja |
| Nequi | Transferencia movil |
| Transferencia | Bancaria |
| Tarjeta | Debito/Credito |
| Credito | Cuenta por cobrar |

---

## Consejos

- El inventario se actualiza automaticamente al vender
- Los cambios y devoluciones se gestionan desde el detalle de la venta
- La contabilidad es global (no por colegio)
- Use el buscador para encontrar productos rapidamente
